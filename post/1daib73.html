Kobe and LeBron 2007-2012 Playoff Numbers Against Same Opponents

The other day, I noticed that Kobe and LeBron played against almost the same opponents in the playoffs from 2007 to 2012, and I wanted to compare their stats. Here are the numbers:

I wish Kobe could have played in the 2013 Spurs series so we could compare that as well.

Lebron vs Spurs 2007: 22.0 points / 7.0 rebounds / 6.8 assists / 35% fg / 42% ts / 37% efg
Kobe vs Spurs 2008: 29.2 points / 5.6 rebounds / 3.8 assists / 53% fg / 58% ts / 56% efg

Lebron vs Celtics 2008: 26.7 points / 6.4 rebounds / 7.6 assists / 35% fg / 48% ts / 38% efg
Kobe vs Celtics 2008: 25.7 points / 4.7 rebounds / 5.0 assists / 40% fg / 50% ts / 43% efg

Lebron vs Magic 2009: 38.5 points / 8.3 rebounds / 8.0 assists / 48% fg / 59% ts / 52% efg
Kobe vs Magic 2009: 32.4 points / 5.6 rebounds / 7.4 assists / 43% fg / 52% ts / 46% efg

Lebron vs Celtics 2010: 26.8 points / 9.3 rebounds / 7.2 assists / 44% fg / 55% ts / 47% efg
Kobe vs Celtics 2010: 28.6 points / 8.0 rebounds / 3.9 assists / 40% fg / 52% ts / 45% efg

Lebron vs Mavs 2011: 17.8 points / 7.2 rebounds / 6.8 assists / 47% fg / 54% ts / 52% efg
Kobe vs Mavs 2011: 23.3 points / 3.0 rebounds / 2.5 assists / 45% fg / 51% ts / 48% efg

Lebron vs OKC 2012: 28.6 points / 10.2 rebounds / 7.4 assists / 47% fg / 55% ts / 48% efg
Kobe vs OKC 2012: 31.2 points / 5.2 rebounds / 3.4 assists / 42% fg / 51% ts / 43% efg
