What if every NBA team went by a state nickname like the Warriors

So I was bored and thought what if every team used a state nickname like the Warriors do with Golden State. All these nicknames I got are from Wikipedia so my apologies if some are wrong. For states with multiple teams I searched for city nicknames (again got them from Wikipedia) . Some of these I didn’t use the most popular nickname because it either didn’t work or some sounded better to me.

Magic City didn’t work for the Heat, Pelican State seemed weird for New Orleans since the team name is the Pelicans, I thought Inland Empire just sounded better than Prairie State for the Bulls, Independence State work better for the Sixers than Keystone State and LA has a plethora of nicknames, but I thought Shaky Town best represented the Clippers.
