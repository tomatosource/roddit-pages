Notes:

  • this is a very crude attempt to put everything together in a single stat. It is calculated as Recovered Blocks 1.2 - Goaltends per game 0.8 - Shooting fouls per game 0.2376
  • Recovered Blocks 1.2 : the average rim shot is around 60% FG and therefore 1.2 PPP. So on average a block is worth 1.2 points ‚Äúsaved‚Äù if recovered
  • Goaltends per game 0.8 : again the average rim shot is already worth 1.2 PPP, so the negative cost of a goaltend is 2 PPP - 1.2 PPP = 0.8 PPP. You are costing your team 0.8 points per goaltend relative to the already high value of a shot at the rim
  • Shooting fouls per game 0.2376 : 0.2376 is 1.760.76 - 1.1 1.76 is the average number of FTs allowed per shooting foul committed (accounting for and 1s), 0.76 is the rough league average FT% over this stretch of NBA history, and 1.1 is roughly the average value of a possession over this stretch. In essence, every time you commit a shooting foul, the opposing team will on average get 0.2376 more points out of it than they would have normally.
  • Wemby being an alien at the component stats makes him an untouchable god here in this attempt to put them all together.
  • Dwight and Javale‚Äôs inefficiencies due to goal tends and foul rates really hurt them here.
  • This number is not meant to represent the per game value of these players as defenders or even just as rim protectors. It obviously doesn‚Äôt account for contested shots, rim deferrence etc‚Ä¶ It‚Äôs just a purely box-derived shot blocking efficiency stat which is very raw.
