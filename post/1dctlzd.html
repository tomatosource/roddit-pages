Despite averaging 9.7 rebounds per game in the playoffs (10th), Luka Doncic has 0.3 box outs per game (68th out of players that average at least 25 mins). Kyrie Irving averages twice as many box outs, and Jayson Tatum has more box outs in 2 finals games than Luka all playoffs.

Source

Now Luka is a guard, so it makes sense that he averages a low amount of box outs. However other, smaller guards (that don’t get nearly as many boards) still rank above him this postseason.

  • Luka Doncic has 0.3 per game
  • Kyrie Irving has 0.6 per game
  • Jrue Holiday has 1.1 per game
  • Donovan Mitchell had 1.3 per game
  • Devin Booker had 1.5 per game
  • Haywood Highsmith had 1.4 per game
  • Kyle Lowry had 1.3 per game
  • CJ McCollum had 1.0 per game
  • Josh Hart had 0.9 per game

Jayson Tatum has had 8 box outs across 2 finals games and Luka Doncic has had 6 box outs over 19 playoff games.

Not all rebounds are created equal. Luka gets a lot of the classic Westbrook uncontested rebounds. For example, in 2017-18 (the first year box outs were tracked) Westbrook averaged 10.1 rebounds per game but had 0.9 box outs per game.
