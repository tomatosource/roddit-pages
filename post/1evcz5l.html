Bam : 436 career blocks, including 66 last year. 436 + 66 = 502 career blocks at the end of the ‘25 season if he has 66 blocks again this year

Wemby : 254 blocks as rookie. 254 + 254 = 508 career blocks at the end of the ‘25 season if he has 254 blocks again this year

I know shotblocking isn’t the main thing Bam does defensively but this is still a pretty wild stat to me.
